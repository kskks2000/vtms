# CLAUDE.md

## 프로젝트 개요

대기업에서 사용하는 **TMS(Transportation Management System, 운송 관리 시스템)** 를 구축하는 프로젝트입니다. 웹과 안드로이드 앱 양쪽에서 사용할 수 있도록 구성합니다.

## 기술 스택

- **프론트엔드**: Flutter (웹 + 안드로이드 앱 동시 지원)
- **백엔드**: FastAPI (Python)
- **데이터베이스**: PostgreSQL 11

> PostgreSQL은 반드시 **버전 11** 기준으로 SQL 문법, 기능, 마이그레이션을 작성합니다. 버전 12 이상에서만 지원되는 기능(예: 일부 generated column, 신규 구문 등)은 사용하지 않습니다.

## 시스템 구성 (모듈)

TMS는 다음 7개 모듈로 구성됩니다.

1. **마스터** — 기준 정보 관리 (거래처, 차량, 기사, 품목, 거점, 운임 등)
2. **오더 생성** — 운송 오더 등록 및 관리
3. **운송계획** — 편성 / 배정 / 배차
4. **실행** — 운송 실행 관리
5. **트래킹** — 차량/화물 위치 추적 (**네이버 지도** 사용)
6. **실적** — 운송 실적 집계 및 관리
7. **정산** — 운임 정산

## 개발 가이드라인

- 신규 기능은 위 7개 모듈 구조에 맞춰 분류하여 개발합니다.
- 지도 연동은 네이버 지도 API를 사용합니다.
- 프론트엔드 코드는 웹과 안드로이드에서 모두 동작하도록 플랫폼 분기를 고려합니다.
- DB 스키마 및 쿼리는 PostgreSQL 11 호환성을 항상 확인합니다.
- API는 RESTful 원칙을 따르며 FastAPI의 자동 문서화(OpenAPI)를 활용합니다.

## 언어

- 사용자와의 커뮤니케이션 및 문서는 **한국어**를 기본으로 합니다.
